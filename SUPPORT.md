# Support

This repository is maintained by a single author.

## What to expect

- Questions, bug reports, and feature requests are welcome.
- Review and response are provided on a best-effort basis.
- There is no formal SLA and no guaranteed release timeline.

## How to ask for help

1. Use GitHub Issues for reproducible bugs and feature suggestions.
2. Provide steps to reproduce, environment details, and logs where possible.
3. For security concerns, follow `SECURITY.md` and use private disclosure.
