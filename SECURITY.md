# Security Policy

## Supported Versions

На текущий момент поддерживается только последняя версия из ветки `main`.

## Reporting a Vulnerability

- Не создавайте публичный issue для уязвимостей.
- Откройте private security advisory в GitHub репозитории.
- Если это недоступно, свяжитесь с maintainers по защищённому каналу и приложите:
  - описание уязвимости;
  - шаги воспроизведения;
  - потенциальное влияние;
  - предложенный mitigation (если есть).

Мы постараемся подтвердить получение отчёта в течение 72 часов и дать план действий после первичного анализа.
